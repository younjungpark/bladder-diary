# Bladder Diary Testing Docs

이 디렉터리는 테스트와 검증 관련 상세 문서를 모아 두는 영역이다.
`docs/` 루트에는 얇은 진입 문서를 두고, 실제 운영 기준과 설명은 이 디렉터리 아래로 모은다.

## 구성

- `smoke-tests.md`
  기본 CLI 검증과 수동 앱 smoke 체크리스트
- `coverage.md`
  테스트 추가 기준과 현재 coverage 운영 방향
- `test-catalog-guide.md`
  현재 테스트 패키지 구성을 빠르게 읽는 방법

## 함께 보는 문서

- `../RELIABILITY.md`
- `../SMOKE_TEST.md`
- `../TEST_COVERAGE.md`
- `../product-specs/sync-behavior.md`

## 운영 원칙

- 테스트 상세 문서는 이 디렉터리에서 관리한다.
- 루트 문서는 빠른 진입과 라우팅에 집중한다.
- 수동 smoke 항목과 자동 테스트 범위를 함께 관리한다.
