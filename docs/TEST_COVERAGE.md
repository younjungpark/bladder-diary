# TEST COVERAGE

이 문서는 테스트 커버리지 빠른 진입 문서다.
상세 기준과 현재 운영 방향은 `testing/coverage.md`에서 관리한다.

## 빠른 확인 순서

1. `.\gradlew.bat :app:testDebugUnitTest`
2. 변경한 패키지에 대응하는 테스트가 있는지 `testing/test-catalog-guide.md`로 확인
3. 새로운 동작이나 회귀 위험이 있으면 단위 테스트를 추가
