# BladderDiary Play Store 등록 문구 초안

이 문서는 `BladderDiary`의 Google Play 등록 시 사용할 설명 문구 초안입니다.

## 1. 앱 이름

- 기본 표기: `BladderDiary`
- 스토어 앱 이름: `BladderDiary`
- 문서/설명 본문: `BladderDiary`

## 2. 짧은 설명 초안

배뇨 기록, 배뇨량·메모 관리, PDF 공유를 지원하는 개인 건강 기록 앱

## 3. 자세한 설명 초안

`BladderDiary`는 배뇨 시간, 배뇨량, 메모를 간편하게 기록하고 필요할 때 PDF로 정리해 공유할 수 있는 개인 건강 기록 앱입니다.

지금 기록과 시간 지정 입력을 지원하며, 날짜별 기록 조회, 배뇨량 입력, 메모 작성, PDF 내보내기, 오프라인 기록 후 자동 동기화를 제공합니다. 병원 방문 전 최근 기록을 개인 참고용으로 정리할 때 활용할 수 있습니다.

`BladderDiary`는 의료기기가 아니며, 진단·치료·응급 판단을 대체하지 않습니다.

### 주요 기능

- 빠른 배뇨 기록 추가
- 시간 지정 기록 추가
- 날짜별 배뇨 기록 조회
- 배뇨량(mL) 입력 및 일일 총 배뇨량 확인
- 메모 작성, 수정, 삭제
- 기간 선택 후 PDF 내보내기 및 공유
- 오프라인 기록 저장 후 온라인 복귀 시 자동 동기화
- Google / Kakao 로그인 지원
- 앱 잠금용 PIN 설정
- 메모 선택형 종단간 암호화(E2EE) 지원

### 이런 분께 적합합니다

- 배뇨 빈도와 배뇨량을 꾸준히 기록하고 싶은 분
- 병원 방문 전 최근 기록을 정리해 가져가고 싶은 분
- 개인 건강 기록을 빠르게 남기고 싶은 분
- 메모 보안과 앱 잠금 기능이 필요한 분

### 보안 및 개인정보 보호

- 기록은 오프라인 상태에서도 기기에 저장됩니다.
- 연결이 가능해지면 자동으로 동기화됩니다.
- 메모는 선택적으로 종단간 암호화(E2EE)할 수 있습니다.
- E2EE를 활성화한 메모는 암호화된 상태로 저장되며, 비밀문구를 분실하면 복구할 수 없습니다.
- PIN은 앱 잠금용 기능이며 E2EE 비밀문구와 목적이 다릅니다.

### PDF 내보내기

- 시작일과 종료일을 선택해 기록 PDF를 만들 수 있습니다.
- 기본적으로 메모는 PDF에 포함되지 않으며, 필요할 때만 옵션으로 포함할 수 있습니다.
- 생성된 PDF는 기기에서 로컬로 만들어지고 사용자가 선택한 외부 앱으로 공유할 수 있습니다.

### 안내 사항

- `BladderDiary`는 의료기기 또는 전문 의료 서비스가 아닙니다.
- 앱의 기록, 통계, PDF 출력물은 개인 참고용이며 진단, 치료, 응급 판단을 대체하지 않습니다.
- 통증, 혈뇨, 발열, 배뇨 곤란 등 이상 증상이 있으면 의료 전문가와 상담해야 합니다.

## 4. 스토어 노출 문구에 넣을 핵심 포인트

- 빠른 배뇨 기록
- 배뇨량 입력
- PDF 공유
- 오프라인 사용 가능
- 자동 동기화
- PIN 잠금
- 메모 선택형 암호화

## 5. 스크린샷 권장 구성

- 메인 기록 화면
- 배뇨량 입력 화면
- 메모 작성 화면
- PDF 내보내기 화면
- PIN 잠금 화면
- E2EE 설정 화면

## 6. 카테고리 및 설정 초안

- 앱 유형: `App`
- 권장 카테고리: `건강/운동`
- 선택 이유: 개인 건강 기록과 추적 중심 앱이므로 전문 의료 도구 성격의 `의료`보다 `건강/운동`이 더 적합
- 광고 포함 여부: `아니오`
- 참고: 스토어 카테고리와 별개로 Play Console의 `Health apps declaration`은 별도로 작성해야 할 수 있음

## 7. 제출 전 점검 메모

- 스토어 설명과 실제 기능이 일치하는지 확인
- PDF 내보내기, 배뇨량, PIN, E2EE가 최신 UI 기준으로 스크린샷에 반영되었는지 확인
- 의료 비고지 안내 문구가 스토어 설명 또는 앱 내 안내와 일치하는지 확인
